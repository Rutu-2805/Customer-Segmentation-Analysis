# Customer-Segmentation-Analysis

Customer segmentation analysis using Python, Clustering, and Power BI to identify customer groups based on purchasing behavior and demographics.

## Project Overview

This project performs customer segmentation using Machine Learning (K-Means Clustering) and Power BI. The goal is to identify distinct customer groups based on purchasing behavior and demographics to support data-driven business decisions.

## Dataset

Dataset: E-Commerce Customer Behavior Dataset

Features include:

* Customer ID
* Gender
* Age
* City
* Membership Type
* Total Spend
* Items Purchased
* Average Rating
* Discount Applied
* Days Since Last Purchase
* Satisfaction Level

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Power BI
* Jupyter Notebook

## Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Feature Scaling
6. Elbow Method
7. K-Means Clustering
8. Customer Segmentation
9. Dashboard Creation in Power BI

## Customer Segments

* Cluster 0: Low spending low-engagement customers
* Cluster 1: Premium high-value customers
* Cluster 2: Budget-conscious occasional buyers
* Cluster 3: Moderate spending regular customers
* Cluster 4: High spending active customers

## Dashboard

The Power BI dashboard contains:

### Customer Overview

* Total Customers
* Average Spend
* Average Rating
* Membership Distribution

### Customer Segments

* Cluster Distribution
* Customer Segmentation Scatter Plot

### Demographics Analysis

* Gender Distribution
* Age Analysis
* Satisfaction Analysis

## Key Insights

* Premium customers contribute the highest spending.
* Customer activity is strongly related to purchase volume.
* Membership type influences customer spending patterns.
* Customer segments can support targeted marketing strategies.

## Results

Successfully segmented customers into meaningful groups using K-Means clustering and visualized customer behavior through an interactive Power BI dashboard.
